This folder contains a modification of the DECOPT solver that enables
it to solve L1-L1 minimization problems.

The source code of the original solver can be found in

  http://lions.epfl.ch/decopt/

The references are

  Q. Tran-Dinh, V. Cevher
  Constrained convex minimization via model-based excessive gap
  Proceedings of the annual conference on Neural Information Processing Systems Foundation (NIPS) 
  2014

  Q. Tran-Dinh, V. Cevher 
  A Primal-Dual Algorithmic Framework for Constrained Convex Minimization
  LIONS Tech. Report EPFL-REPORT-199844 
  2014

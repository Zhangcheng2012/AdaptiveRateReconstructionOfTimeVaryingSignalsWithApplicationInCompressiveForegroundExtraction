This folder contains the code described in the papers

  G. Warnell, S. Bhattacharya, R. Chellappa, T. Basar
  "Adaptive-Rate Compressive Sensing via Side Information" 
  http://arxiv.org/abs/1401.0583
  2014

  G. Warnell, D. Reddy, and R. Chellappa
  "Adaptive-Rate Compressive Sensing for Background Subtraction" 
  Intern. Conf. Acoustics, Speech, and Signal Processing (ICASSP)
  2012
  
The original source code can be found in http://garrettwarnell.com/ARCS-1.0.zip
